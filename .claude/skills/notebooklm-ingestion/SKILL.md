# NotebookLM Ingestion Skill

> Extracts content from a Google NotebookLM notebook into standardized files for the dashboard pipeline.

## Purpose

This skill handles Step 2 (Content Extraction) when `source_type` is `notebooklm`. It connects to NotebookLM via the `notebooklm-py` library, extracts source fulltexts and a generated Study Guide, and produces the same output structure expected by the rest of the pipeline.

## Prerequisites

The user must have authenticated once via:
```bash
notebooklm login
```
Credentials are stored by notebooklm-py in `~/.notebooklm/` and reused automatically.

## Supported Input

- **Notebook URL**: `https://notebooklm.google.com/notebook/<id>`
- **Notebook ID**: bare alphanumeric ID string

## Usage

```bash
python .claude/skills/notebooklm-ingestion/scripts/extract_notebooklm.py "<notebook_url_or_id>" "<output_dir>"
```

## Output Files

| File | Description | Required |
|------|-------------|----------|
| `raw_text.md` | Combined fulltext from all sources (Part A) | Always |
| `notebooklm_report.md` | Study Guide generated by NotebookLM (Part B) | Best-effort |
| `source_metadata.json` | Notebook title, source count, source URLs | Always |

### raw_text.md format

- Single source: raw fulltext
- Multiple sources (>1): TOC header + each source separated by `---`
- Maximum 10 sources processed (warning printed if more exist)

### notebooklm_report.md

- Markdown Study Guide generated (or fetched if already exists) via NotebookLM
- Passed to content-analyzer as a structural hint alongside `raw_text.md`
- If generation fails, the pipeline continues with `raw_text.md` only

### source_metadata.json fields

```json
{
  "notebook_id": "...",
  "notebook_title": "...",
  "notebook_url": "https://notebooklm.google.com/notebook/...",
  "source_count": 3,
  "sources_processed": 3,
  "sources": [
    { "title": "...", "url": "...", "index": 1 }
  ]
}
```

## Validation

After extraction, validate using the shared validation script:
```bash
python .claude/skills/content-ingestion/scripts/validate_extraction.py "<output_dir>/raw_text.md"
```

## Error Handling

| Error | Cause | Resolution |
|-------|-------|------------|
| `Not authenticated` | `notebooklm login` not run | Run `notebooklm login` |
| `Could not access notebook` | Wrong ID or no permission | Check the notebook URL/ID |
| `Could not extract text from any sources` | All sources have no fulltext | Check sources in NotebookLM UI |
| `Could not generate Study Guide` | API error | Pipeline continues without report |

## Dependencies

```
notebooklm-py
```

Install:
```bash
pip install notebooklm-py
```
