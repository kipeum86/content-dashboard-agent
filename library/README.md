## Registration Instructions

To add a dashboard to the library:
1. Complete and review a dashboard
2. Tell the agent: "add to library"
3. The agent will copy the HTML and update this table

## Naming Convention

```
example-<content_type>-<sequence>.html
```

Examples: `example-document-01.html`, `example-media-02.html`, `example-book-01.html`
